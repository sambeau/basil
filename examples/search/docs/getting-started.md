# Getting Started with Basil

Welcome to Basil! This guide will help you get started quickly.

## Installation

Install Basil using Go:

```bash
go install github.com/sambeau/basil/cmd/basil@latest
```

## Creating Your First Project

Create a new directory for your project:

```bash
mkdir my-basil-app
cd my-basil-app
```

## Running the Server

Start the Basil server:

```bash
basil serve
```

Visit http://localhost:8080 to see your app!
