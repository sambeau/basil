# Basil

A Go (Golang) project.

## Prerequisites

- [Go](https://golang.org/dl/) 1.21 or later

## Getting Started

### Build

```bash
go build -o basil .
```

### Run

```bash
go run .
```

Or after building:

```bash
./basil
```

### Test

```bash
go test ./...
```
